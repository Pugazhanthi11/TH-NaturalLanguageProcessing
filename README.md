# Natural Language Processing (NLP)

For this, I explored a basic Natural Language Processing (NLP) application using customer reviews.

I performed sentiment analysis on a small set of text reviews and classified them as positive, negative, or neutral based on their sentiment.

## Tools Used

- Python
- Pandas
- TextBlob

## Files

- nlp_analysis.py
- sentiment_results.csv

## What I Learned

This introduced me to the basics of NLP and sentiment analysis. It showed how text data can be processed and analyzed to understand opinions and emotions expressed in reviews.
